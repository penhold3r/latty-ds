# Design Tokens

Design tokens are the foundation of the Latty Design System. They are the single source of truth for design decisions, generating CSS custom properties, JSON exports, and TypeScript types.

## Token Categories

Latty provides tokens in the following categories:

### Colors

Complete color palettes with tints and shades automatically generated from base colors:

- **Primary** - `--lt-color-primary-{50-950}`
- **Secondary** - `--lt-color-secondary-{50-950}`
- **Success** - `--lt-color-success-{50-950}`
- **Warning** - `--lt-color-warning-{50-950}`
- **Error** - `--lt-color-error-{50-950}`
- **Neutral** - `--lt-color-neutral-{50-950}`

### Spacing

Two scales for spacing tokens:

- **rem-based** - `--lt-spacing-{1-12}` (default)
- **px-based** - `--lt-spacing-px-{1-12}`

### Typography

Typography tokens for consistent text styling:

- **Font families** - `--lt-font-family-*`
- **Font sizes** - `--lt-font-size-*`
- **Line heights** - `--lt-line-height-*`
- **Font weights** - `--lt-font-weight-*`

### Other Tokens

- **Border radius** - `--lt-border-radius`
- **Shadows** - `--lt-shadow-*`
- **Z-index** - `--lt-z-index-*`

## Using Tokens

Tokens are available as CSS custom properties after importing the CSS file:

```css
@import '@latty/tokens/tokens.css';

.my-component {
  color: var(--lt-color-primary-500);
  padding: var(--lt-spacing-4);
  border-radius: var(--lt-border-radius);
}
```

## Customization

Override tokens to create custom themes:

```css
:root {
  /* Custom primary color */
  --lt-color-primary-500: #8b5cf6;
  
  /* Custom spacing scale */
  --lt-spacing-4: 2rem;
  
  /* Custom border radius */
  --lt-border-radius: 8px;
}
```

## JavaScript/TypeScript Usage

Import tokens as JavaScript/TypeScript objects:

```typescript
import tokens from '@latty/tokens';

const primaryColor = tokens.color.primary[500];
const spacing = tokens.spacing.rem[4];
```
