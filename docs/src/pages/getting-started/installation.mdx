# Installation

## Install packages

```bash
# pnpm
pnpm add @latty/tokens @latty/web

# npm
npm install @latty/tokens @latty/web

# yarn
yarn add @latty/tokens @latty/web
```

`@latty/icons` is bundled inside `@latty/web` — you don't need to install it separately unless you want to use the icon registry API directly.

## Import tokens

The tokens CSS file must be loaded before any components render. Import it once at your app's entry point:

```js
import '@latty/tokens/dist/tokens.css';
```

With a CSS `@import` (e.g. in a global stylesheet):

```css
@import '@latty/tokens/dist/tokens.css';
```

In a plain HTML file:

```html
<link rel="stylesheet" href="node_modules/@latty/tokens/dist/tokens.css">
```

## Import components

```js
// All components at once
import '@latty/web';

// Or cherry-pick to keep bundles small
import '@latty/web/button';
import '@latty/web/textfield';
import '@latty/web/dialog';
```

That's it — components are self-registering custom elements. Once imported, use them anywhere in your HTML.

## Framework wrappers (optional)

For typed React components:

```bash
pnpm add @latty/react
```

Vue users don't need a wrapper package — `lt-*` elements work natively in Vue 3 templates. See the [Usage guide](/getting-started/usage) for the one-time setup.

## TypeScript

All packages ship with `.d.ts` files. No extra setup needed.

To get HTML template type-checking for custom elements in TypeScript projects, add this to your `tsconfig.json`:

```json
{
  "compilerOptions": {
    "lib": ["ES2020", "DOM", "DOM.Iterable"]
  }
}
```
