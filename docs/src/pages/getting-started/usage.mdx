# Usage

## Vanilla HTML

The fastest way to get started — no build tool required:

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <link rel="stylesheet" href="node_modules/@latty/tokens/dist/tokens.css">
</head>
<body>
  <lt-button variant="primary" icon="save">Save changes</lt-button>

  <script type="module">
    import '@latty/web';
  </script>
</body>
</html>
```

## With a bundler (Vite, Webpack, etc.)

```js
// entry.js
import '@latty/tokens/dist/tokens.css';
import '@latty/web';
```

```html
<lt-button variant="primary">Click me</lt-button>

<lt-textfield
  label="Email address"
  type="email"
  required
  helper-text="We'll never share your email"
></lt-textfield>
```

## React

```tsx
import '@latty/tokens/dist/tokens.css';
import '@latty/web';

// Web Components work directly in JSX — no wrapper needed
export function SaveButton() {
  return (
    <lt-button
      variant="primary"
      icon="save"
      onClick={() => console.log('saved')}
    >
      Save changes
    </lt-button>
  );
}
```

For full TypeScript prop types and idiomatic React events, use the `@latty/react` wrapper package instead.

## Angular

Add `CUSTOM_ELEMENTS_SCHEMA` to any module or standalone component that uses `lt-` elements:

```typescript
import { Component, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';
import '@latty/tokens/dist/tokens.css';
import '@latty/web';

@Component({
  selector: 'app-root',
  template: `
    <lt-button variant="primary" (click)="save()">Save</lt-button>
  `,
  schemas: [CUSTOM_ELEMENTS_SCHEMA],
})
export class AppComponent {
  save() { /* ... */ }
}
```

## Listening to events

Components dispatch standard DOM events:

```js
const dialog = document.querySelector('lt-dialog');

dialog.addEventListener('lt-open', () => console.log('opened'));
dialog.addEventListener('lt-close', () => console.log('closed'));
```

```js
const textfield = document.querySelector('lt-textfield');

textfield.addEventListener('lt-change', (e) => {
  console.log(e.detail.value);
});
```

## Setting properties programmatically

Attributes handle strings and booleans. For richer data (e.g. table columns), use the JS property directly:

```js
const table = document.querySelector('lt-table');
table.columns = [
  { key: 'name', label: 'Name' },
  { key: 'email', label: 'Email' },
];
table.data = [
  { name: 'Alice', email: 'alice@example.com' },
];
```
